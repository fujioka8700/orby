## 3. ゲームの再開方法の演出（案）

| 思い付いた日 | 実装した日 |
| ------------ | ---------- |
| 2025-01-31   |            |

ゲームオーバー後、プレイヤーがどうやってもう一度遊ぶか。

### 3.1 決めること

| 項目             | 候補・メモ                                                               |
| ---------------- | ------------------------------------------------------------------------ |
| **操作**         | 「スペースキー／Enter／Z で再開」／「タップ・クリックで再開」／ボタン UI |
| **画面表示**     | 「PRESS SPACE TO RESTART」など再開方法を GAME OVER の下に表示            |
| **再開時の状態** | シーンを再読み込み（ステージ・残機を最初から）／タイトルへ戻す           |
| **演出**         | フェードアウト → シーン再開 → フェードイン／即座に再開                   |

### 3.2 暫定方針（実装の目安）

- **再開方法**：GAME OVER 画面に「PRESS SPACE TO RESTART」または「スペースキーでやり直し」を表示。
- **操作**：スペースキー（または Enter）で「ゲームオーバー状態」を解除し、シーンを再起動（`scene.restart()`）してステージ・残機を最初からに戻す。
- **演出**：まずは「キー入力で即シーン再開」。余裕があればフェードを挟む。
